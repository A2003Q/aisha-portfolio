# Aisha Qarout Portfolio Website

Personal portfolio website for Aisha Qarout, Full Stack Web Developer.

## Featured Projects

- Parking Finder — Laravel, PHP, MySQL, JavaScript
- Gold Price Tracker — Laravel API, React, External APIs
- E-Commerce Clothes Store — PHP, JavaScript, MySQL

## How to run locally

Open `index.html` in your browser.

## Deploy on GitHub Pages

1. Create a new GitHub repository named `aisha-portfolio` or `portfolio`.
2. Upload all files from this folder, not the ZIP file itself.
3. Go to repository Settings → Pages.
4. Under Branch, select `main` and `/root`, then Save.
5. Wait 1–3 minutes and GitHub will give you your public portfolio link.
