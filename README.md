# Aisha Qarout Portfolio

Static portfolio website built with HTML, CSS, and JavaScript.

## Deploy on GitHub Pages

1. Upload all files and folders to the repository root.
2. Make sure `index.html` is in the root, not inside another folder.
3. Go to Settings > Pages.
4. Choose Deploy from branch.
5. Branch: main, Folder: /root.
6. Save and wait a few minutes.
